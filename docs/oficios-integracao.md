## OFÍCIO 1 — SIAT (Secretaria Municipal de Economia)

```
OFÍCIO Nº [XXX]/2026 – SMS/VISA

Maceió, [dia] de [mês] de 2026.

À
SECRETARIA MUNICIPAL DE ECONOMIA – SEMEC
Divisão de Tecnologia da Informação
Maceió – AL

Assunto: Solicitação de acesso à API/Webservice do SIAT para integração
         com o Sistema de Licenciamento Sanitário Municipal.

Senhor(a) Secretário(a),

A Secretaria Municipal de Saúde, por intermédio da Diretoria de
Vigilância Sanitária, vem respeitosamente solicitar a disponibilização
de credenciais de acesso (endpoint, token de autenticação e documentação
técnica) ao Sistema de Informações de Arrecadação Tributária – SIAT,
para fins de integração com o novo Sistema de Licenciamento Sanitário
Municipal.

A referida integração tem como objetivo:
  • Consultar a situação fiscal de estabelecimentos por CNPJ;
  • Verificar a existência de débitos em aberto;
  • Emitir Certidão Negativa de Débitos de forma automatizada.

Essa automação permitirá maior celeridade na análise de processos de
licenciamento sanitário, reduzindo o prazo de atendimento ao cidadão e
garantindo que apenas estabelecimentos adimplentes obtenham a licença.

Informamos que o sistema já se encontra preparado tecnicamente para
receber a integração, bastando a configuração das credenciais fornecidas.

O responsável técnico pelo sistema é [NOME DO RESPONSÁVEL TÉCNICO],
e-mail: [email@prefeitura.gov.br], telefone: [(__) _____-____].

Atenciosamente,

_________________________________
[NOME DO SECRETÁRIO DE SAÚDE]
Secretário(a) Municipal de Saúde
Maceió – AL
```

## OFÍCIO 2 — SUPE (SEPLAG/SEFAZ Alagoas)

[ou SECRETARIA DE ESTADO DO PLANEJAMENTO – SEPLAG/AL]
Maceió – AL

Assunto: Solicitação de acesso à API do Sistema Único de Pagamentos
         Eletrônicos – SUPE para emissão de guias DAM.

Vigilância Sanitária, solicita a disponibilização de credenciais de

Solicitamos o envio de:
  1. URL base da API / Webservice;
  2. Credenciais de acesso (API Key, token ou usuário/senha);
  3. Documentação técnica da API (endpoints, formatos, exemplos).

O responsável técnico é [NOME], contato: [email], [telefone].

Atenciosamente,

_________________________________
[NOME DO SECRETÁRIO DE SAÚDE]
Secretário(a) Municipal de Saúde
Maceió – AL

## OFÍCIO 3 — Facilita Alagoas / JUCEAL

```
OFÍCIO Nº [XXX]/2026 – SMS/VISA

Maceió, [dia] de [mês] de 2026.

À
JUNTA COMERCIAL DO ESTADO DE ALAGOAS – JUCEAL
Maceió – AL

Assunto: Solicitação de convênio/acesso à base de dados da JUCEAL via
         Portal Facilita Alagoas para integração com Sistema de
         Licenciamento Sanitário.

Senhor(a) Presidente,

A Secretaria Municipal de Saúde de Maceió vem solicitar a celebração
de acordo de cooperação técnica ou convênio para acesso à base de
registros empresariais da JUCEAL, disponibilizada por meio do Portal
Facilita Alagoas.

A integração tem como finalidade:
  • Consultar o registro e a situação cadastral de empresas (NIRE);
  • Verificar se o estabelecimento possui registro ativo na JUCEAL
    antes da concessão de licença sanitária;
  • Vincular o número de registro JUCEAL ao cadastro do estabelecimento
    no sistema municipal.

Essa medida está alinhada com as diretrizes da RedeSIM e da Lei de
Liberdade Econômica (Lei nº 13.874/2019), que preconizam a integração
entre órgãos para simplificação do licenciamento.

Solicitamos orientação sobre:
  1. Procedimento para formalização do acesso;
  2. Disponibilidade de API ou webservice para consulta;
  3. Documentação técnica, se aplicável.

Contato técnico: [NOME], [email], [telefone].

Atenciosamente,

_________________________________
[NOME DO SECRETÁRIO DE SAÚDE]
Secretário(a) Municipal de Saúde
Maceió – AL
```

## OFÍCIO 4 — Corpo de Bombeiros Militar de Alagoas (CBMAL)

```
OFÍCIO Nº [XXX]/2026 – SMS/VISA

Maceió, [dia] de [mês] de 2026.

Ao
CORPO DE BOMBEIROS MILITAR DO ESTADO DE ALAGOAS – CBMAL
Setor de Prevenção e Proteção contra Incêndio
Maceió – AL

Assunto: Solicitação de integração para consulta de laudos AVCB/CLCB
         no Sistema de Licenciamento Sanitário Municipal.

Senhor Comandante,

A Secretaria Municipal de Saúde, por meio da Vigilância Sanitária,
solicita a celebração de acordo de cooperação técnica para integração
entre o Sistema de Licenciamento Sanitário Municipal e o sistema de
controle de vistorias do CBMAL.

O objetivo da integração é:
  • Consultar automaticamente o status de AVCB (Auto de Vistoria do
    Corpo de Bombeiros) ou CLCB (Certificado de Licenciamento do Corpo
    de Bombeiros) dos estabelecimentos;
  • Verificar a validade do documento antes da concessão da licença
    sanitária;
  • Condicionar a aprovação de licenças de atividades de alto risco à
    apresentação de AVCB/CLCB válido.

Essa integração é fundamental para garantir a segurança dos
estabelecimentos licenciados e está prevista no edital de contratação
do sistema (conforme exigências do Termo de Referência).

Solicitamos:
  1. Viabilidade técnica de disponibilização de API ou webservice;
  2. Credenciais de acesso para ambiente de homologação/produção;
  3. Documentação técnica dos serviços disponíveis.

Contato técnico: [NOME], [email], [telefone].

Atenciosamente,

_________________________________
[NOME DO SECRETÁRIO DE SAÚDE]
Secretário(a) Municipal de Saúde
Maceió – AL
```

## Após Receber as Credenciais

Configure no arquivo `.env.local` do projeto:

```env
# SIAT
SIAT_API_URL=https://...
SIAT_API_KEY=...

# SUPE
SUPE_API_URL=https://...
SUPE_API_KEY=...

# Facilita / JUCEAL
FACILITA_API_URL=https://...
FACILITA_API_KEY=...

# Bombeiros
BOMBEIROS_API_URL=https://...
BOMBEIROS_API_KEY=...
```
